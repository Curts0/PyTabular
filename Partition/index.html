.Refresh(

Returns

  • DataFrame : Returns pandas dataframe with some refresh details

PyPartitions

source

PyPartitions(
   objects
)

Groups together multiple partitions. See PyObjects class for what more it can do. You can interact with PyPartitions straight from model. For ex: model.Partitions. Or through individual tables model.Tables[TABLE_NAME].Partitions. You can even filter down with .Find(). For example find all partition with prev-year in name. model.Partitions.Find('prev-year').
