Base_Trace

source

Base_Trace(
   Tabular_Class, Trace_Events: List[TraceEvent],
   Trace_Event_Columns: List[TraceColumn], Handler: Callable
)

Generates Trace to be run on Server. This is the base class to customize the type of Trace you are looking for.
Server Traces

Args

  • Tabular_Class (Tabular) : Tabular Class to retrieve the connected Server and Model.
  • Trace_Events (List[TraceEvent]) : List of Trace Events that you wish to track. TraceEventClass
  • Trace_Event_Columns (List[TraceColumn]) : List of Trace Event Columns you with to track. TraceEventColumn
  • Handler (Callable) : Function to call when Trace returns response. Input needs to be two arguments. One is source (Which is currently None... Need to investigate why). Second is TraceEventArgs
