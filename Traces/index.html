BaseTrace

source

BaseTrace(
   tabular_class, trace_events: List[TraceEvent],
   trace_event_columns: List[TraceColumn], handler: Callable
)

Generates Trace to be run on Server. This is the base class to customize the type of Trace you are looking for. Server Traces

Args

  • tabular_class (Tabular) : Tabular Class.
  • trace_events (List[TraceEvent]) : List of Trace Events.
  • trace_event_columns (List[TraceColumn]) : List of Trace Event Columns.
  • Handler (Callable) : Function to call when Trace returns response. TraceEventClass TraceEventColumn Input needs to be two arguments. One is source (Which is currently None... Need to investigate why). Second is TraceEventArgs

Methods:

  • bool : True if successful
